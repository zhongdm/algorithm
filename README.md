+ 树
  + 二叉树： BST（**左节点必须小于右节点**）， 
    + 中序遍历： 左 -> 根 -> 右
    + 先序遍历： 根 -> 左 -> 右
    + 后序遍历： 左 -> 右 -> 根
    + AVL树，红黑树 （**没理解**）
    ```
      每个节点要么是黑的，要么是红的
      根节点是黑的
      叶节点是黑的
      如果一个节点是红的，他的两个儿子节点都是黑的
      对于任一节点而言，其到叶节点树尾端NIL指针的每一条路径都包含相同数目的黑节点
    ```
